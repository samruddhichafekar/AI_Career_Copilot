# 🤖 AI Career Copilot

AI Career Copilot is an AI-powered web application that helps job seekers analyze their resumes, identify skill gaps, and receive personalized career guidance based on their target job role.

The application allows users to upload their resume, select a desired career path, and receive AI-generated insights such as relevant skills, missing skills, learning roadmap, and interview questions.

---

## 🚀 Features

- 🔐 User Authentication (Sign Up/Login)
- 📄 Resume Upload (PDF & DOCX)
- 📝 Manual Resume Text Input
- 🤖 AI-Powered Resume Analysis
- 🎯 Career Goal Based Recommendations
- 📊 Skill Gap Analysis
- 📚 Personalized Learning Roadmap
- ❓ AI-Generated Interview Questions
- 🗂 Analysis History
- 💾 PostgreSQL Database Integration
- 🌐 Responsive Flask Web Application
